# Screenshots for Mathcad Automator Documentation

This directory contains screenshots for the README.md documentation.

## Required Screenshots

### 1. Main Interface (`screenshot-main.png`)

**What to capture:**
- The main application window showing:
  - Batch Processing tab with a file loaded
  - Input configuration panel visible
  - Several input aliases listed

**How to capture:**
1. Open MathcadAutomator.exe
2. Click Browse and select a .mcdx file with multiple inputs
3. Click on one input to show the configuration modal
4. Use Windows Snipping Tool (Windows key + Shift + S)
5. Save as `screenshot-main.png`

**Caption:** "Main interface showing file selection and input configuration"

### 2. Results View (`screenshot-results.png`)

**What to capture:**
- A completed batch operation showing:
  - Progress bar at 100%
  - Results list with generated files
  - Several PDF/MCDX files listed

**How to capture:**
1. Run a small batch (5-10 iterations)
2. Wait for completion
3. Capture the results section
4. Save as `screenshot-results.png`

**Caption:** "Completed batch with generated files listed"
